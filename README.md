 Description:
 A dual-interface system developed in C++ that distinguishes access between Users and Administrators.
 User Interface: Allows students to view available facilities, book schedules, and monitor their own booking status.
 Admin Interface: Provides full control for administrators to manage facilities, approve or reject booking requests, and update facility availability.
 
 Key Features:
 Secure role-based access control (RBAC).
 Real-time validation to prevent double-booking of facilities.
